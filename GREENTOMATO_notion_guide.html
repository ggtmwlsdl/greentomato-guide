저작권 안전
📋
진행 순서 5단계
1
주제 & 아이디어 정하기
Claude에게 내 분야 카드뉴스 주제 30개를 요청합니다. 그 중 반응이 좋을 것 같은 주제 하나를 고르세요.
🤖 Claude 사용
2
슬라이드 문구 초안 작성
선택한 주제로 Claude에게 슬라이드 5~6장 분량의 문구를 요청합니다. 제목 1줄 + 설명 2줄 구성으로 요청하면 바로 쓸 수 있는 형태로 나옵니다.
🤖 Claude 사용
3
Canva에서 디자인
Canva → 새 디자인 → 1080×1350px 설정 후 문구를 붙여넣고 색상·폰트를 적용합니다. Pretendard ExtraBold(제목) + SemiBold(본문)를 추천합니다.
🎨 Canva 사용
4
게시물 설명 & 해시태그 작성
Claude에게 완성된 카드뉴스 주제를 알려주고 인스타그램 게시물 설명과 해시태그를 요청합니다. 정보형 / 공감형 2가지 버전을 받아서 선택하세요.
🤖 Claude 사용
5
PNG 다운로드 & 인스타 업로드
Canva → 다운로드 → PNG → 인스타그램에 순서대로 6장 업로드. 최적 업로드 시간대는 오전 7~9시 또는 저녁 7~9시입니다.
📱 인스타그램 업로드
💬
프롬프트 모음
① 주제 아이디어 뽑기 기획
[내 분야/업종]을 운영하는 인스타그램 계정을 위한
카드뉴스 주제 30개를 뽑아줘.
저장율이 높고 공유하고 싶어지는 주제 위주로,
각 주제는 한 줄로 간결하게 작성해줘.
② 슬라이드 문구 작성 제작
주제: [주제 입력]
위 주제로 인스타그램 카드뉴스 6장 분량의 문구를 써줘.
각 슬라이드는 아래 형식으로:
- 제목: 1줄 (10자 이내)
- 부제목: 1줄
- 본문: 2~3줄
마지막 장은 저장 유도 CTA로 마무리해줘.
③ 카드뉴스 디자인 시각화 디자인
아래 내용으로 인스타그램 카드뉴스를 만들어줘.
- 사이즈: 1080 × 1350px
- 폰트: Pretendard ExtraBold(제목), SemiBold(본문)
- 브랜드명: @GREENTOMATO
- 슬라이드 수: 6장
- 업로드 주기: 매주 월·수·금
[슬라이드 내용 붙여넣기]
④ 게시물 설명 작성 업로드
주제: [주제 입력]
위 카드뉴스를 인스타그램에 올릴 때 쓸
게시물 설명을 2가지 버전으로 써줘.
- 버전1: 정보 전달형 (저장 유도)
- 버전2: 공감형 (감성 + 저장 유도)
해시태그 15개도 포함해줘.
⑤ 시리즈 콘텐츠 기획 기획
[주제/카테고리]를 큰 테마로,
이어지는 느낌의 4부작 시리즈 카드뉴스를 기획해줘.
각 편의 제목, 핵심 내용, 어떤 순서로 이어지는지
플로우를 설명해줘.
화~금 요일에 업로드하는 일정으로 맞춰줘.
⑥ 콘텐츠 재활용 전략 운영
아래 카드뉴스 내용을 기반으로
다양한 포맷으로 재활용할 수 있는 방법 5가지를 알려줘.
(스토리, 릴스, 브로드캐스트, 블로그 등)
각 포맷별 핵심 수정 포인트도 함께 알려줘.
[카드뉴스 내용 붙여넣기]
⑦ 한 달 치 콘텐츠 캘린더 기획
[내 분야/업종] 인스타그램 계정을 위한
한 달 치 콘텐츠 캘린더를 만들어줘.
- 업로드 주기: 월·수·금 (주 3회)
- 포맷: 카드뉴스 위주
- 시리즈물과 단독 게시물을 섞어서 구성
날짜별 주제와 간단한 방향성을 표로 정리해줘.
✅
업로드 전 체크리스트
0 / 15 완료
📝 기획 단계
✓
주제가 명확하게 하나로 좁혀져 있다
한 슬라이드 = 한 메시지
✓
타겟 독자가 누군지 정해져 있다
20대 직장인, 소상공인, 육아맘 등
✓
슬라이드 수가 5~7장 사이다
너무 많으면 끝까지 안 넘겨요
✍️ 문구 단계
✓
표지 제목이 한눈에 읽힌다 (10자 이내)
스크롤 중 멈추게 만드는 첫 장
✓
슬라이드당 글자 수가 50자 이하다
짧을수록 더 잘 읽혀요
✓
마지막 장에 저장 유도 문구가 있다
"저장해두면 나중에 도움돼요" 류
✓
브랜드명 @GREENTOMATO가 들어가 있다
표지 또는 마지막 장에 필수
🎨 디자인 단계
✓
캔버스 사이즈가 1080 × 1350px다
인스타 세로형 최적 비율
✓
사용 색상이 2~3가지 이내다
색이 많을수록 산만해 보여요
✓
제목 폰트가 ExtraBold로 크게 설정됐다
Pretendard ExtraBold 권장
✓
6장의 디자인 톤이 통일됐다
색상·폰트·여백이 일관적
✓
PNG로 다운로드했다 (JPG X)
PNG가 텍스트 선명도 훨씬 좋아요
📱 업로드 단계
✓
슬라이드 순서가 맞게 정렬됐다
표지 → 본문 → CTA 순서
✓
게시물 설명에 해시태그 10~15개 포함됐다
관련 키워드 + 브랜드 해시태그
✓
최적 시간대에 업로드한다
오전 7~9시 또는 저녁 7~9시 추천
💡
알아두면 좋은 꿀팁
⏱

주 1회 몰아서 만들고 예약 업로드. Claude로 주제 뽑기 → 문구 작성 → Canva 디자인 → 예약
